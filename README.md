# Numerica Studio

Numerica Studio is a scientific computing web application built with Streamlit.  
The application provides a unified environment for mathematical computation, visualization, and matrix analysis, with a focus on supporting tasks in machine learning and deep learning.

## Features
- Scientific calculator for mathematical expressions
- Function plotting in both 2D and 3D
- Matrix algebra operations
- Singular Value Decomposition (SVD)
- Principal Component Analysis (PCA)
